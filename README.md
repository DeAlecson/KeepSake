# Keepsake

> Timeless resin keepsakes — necklaces, keychains, and bespoke pet-hair memorials.

The "Eternal Bond" website MVP. Static HTML/CSS/JS — no build step required.

## Pages

| File | Purpose |
| --- | --- |
| `index.html` | Homepage — hero carousel, bespoke section, Product 1 shop grid, Feature X teaser, footer |
| `customize.html` | Product 2 customizer — 6-step bespoke flow (template → addons → attachment → review → payment → instructions) |
| `feature-x.html` | NFC pet registry — registration flow + community pet grid |
| `handoff.html` | Engineering handoff spec from Claude Design |

## Products

- **Product 1** — Predesigned, tri-color resin pucks. No customisation. Limited drops (10 necklaces / 5 keychains per drop). Styles: `lava`, `goldflake`, `flora`, `colorwash`, `marble`. SVGs generated by `drop1-svg.js`.
- **Product 2** — Bespoke pet-hair keepsakes with NFC chip. Customizable templates, add-ons, and attachments. SVGs generated by `puck-svg.js`.

## Design system

The "Eternal Bond" palette:

| Token | Hex | Use |
| --- | --- | --- |
| `--navy` | `#0F172A` | Anchor / heritage |
| `--gold` | `#C69B3C` | Accent / CTAs |
| `--ivory` | `#F4EADE` | Background |
| `--taupe` | `#8A7968` | Secondary neutral |

Fonts: `Cormorant Garamond` (display) + `DM Sans` (body), via Google Fonts.

## Run locally

Open `index.html` directly in a browser, or serve the folder:

```bash
python -m http.server 8000
# or
npx serve .
```
